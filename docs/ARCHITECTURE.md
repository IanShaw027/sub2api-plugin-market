# Sub2API Plugin Market - 架构设计

本文档描述 Sub2API Plugin Market 的系统架构、设计决策和技术实现。

---

## 系统概览

Sub2API Plugin Market 是一个插件市场服务，提供插件的浏览、下载和签名验证功能。

### 核心功能

1. **插件浏览** - 支持分类、搜索、分页
2. **版本管理** - 多版本支持，语义化版本控制
3. **安全下载** - Ed25519 签名验证
4. **信任密钥** - 公钥管理和验证
5. **存储管理** - MinIO 对象存储

## 技术决策

### 1. 为什么选择 Gin？

- **性能**: 高性能 HTTP 路由，基于 httprouter
- **简洁**: API 设计简洁，易于上手
- **中间件**: 丰富的中间件生态
- **社区**: 活跃的社区支持

### 2. 为什么选择 Ent？

- **类型安全**: 强类型 ORM，编译时检查
- **代码生成**: 自动生成查询代码
- **图遍历**: 支持复杂的关联查询
- **迁移**: 内置迁移工具

### 3. 为什么选择 Ed25519？

- **安全**: 现代椭圆曲线签名算法
- **性能**: 签名和验证速度快
- **密钥**: 密钥长度短（32 字节）
- **标准**: Go 标准库支持

### 4. 为什么选择 MinIO？

- **兼容**: S3 API 兼容
- **开源**: 完全开源，可自托管
- **性能**: 高性能对象存储
- **易用**: 部署和管理简单

└──────┬──────┘

       ▼

       │
       ▼

## 性能优化

### 1. 缓存策略

- **Redis 缓存**: 插件列表、插件详情
- **缓存时间**: 5 分钟
- **缓存失效**: 插件更新时主动失效

### 2. 数据库优化

- **索引**: name, category, is_official
- **连接池**: 最大 100 连接
- **预加载**: 使用 Ent 的 `With` 方法
- **分页**: 限制最大 100 条/页

### 3. 存储优化

- **CDN**: 静态文件 CDN 加速
- **压缩**: WASM 文件 gzip 压缩
- **流式传输**: 大文件流式下载

- **结构化日志**: JSON 格式

- **业务指标**: 下载次数, 活跃插件数

### 3. 追踪

- **请求 ID**: 全链路追踪
- **日志关联**: 通过请求 ID 关联日志
